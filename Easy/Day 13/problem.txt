Problem Statment
Given an integer N, check whether N is a palindrome or not.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Input

Integer N.
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Output
Return 1 if N is a palindrome, and -1 otherwise.

-----------------------------------------------------------------------------------------------------------------------------------------------
-------------------------

Constraints:
-2^31 <= x <= 2^31 - 1


-----------------------------------------------------------------------------------------------------------------------------------------------
-------------------------

Time limit - 1 sec
